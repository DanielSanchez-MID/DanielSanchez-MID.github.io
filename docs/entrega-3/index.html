Mecanizado CNC parte 2: Respaldo

Encargo 3

Respaldo Silla Fresia en Fusion 360

Proceso en el software y pasos posteriores

En el caso de la Silla Fresia, el respaldo es curvo, por lo que ya no solo trabajaremos con procesos 2D en Fusion 360, sino que recurriremos a procesos de tipo 3D (2.5D). Mas adelante veremos por que se le llama 2.5D aun que este en el menu de mecanizados 3D. Para esto, utilizamos la misma máquina que en la publicación anterior (generic 3 axis).

A continuación se mostrará documentación sobre el proceso en software que se debe hacer para poder hacerla con un mecanizado en CNC. Además se mostrará a través des d fotografías el proceso posterior por el que debe pasar el respaldo después de salir de la máquina.

Desbaste

Se refiere a la acción de eliminar fragmentos grandes de material que no sera utilizado para la pieza final y que se debe eliminar para trabajar con mayor detalle. En este caso, el respaldo tiene una curva, por lo que todo el material que hay entre el exterior de la plancha y el respaldo en si, debe ser limpiado. A este proceso se le llama desbaste.

Paso 2

A continuación se hace una limpieza o desbaste adaptativo. Esto significa que se quita todo el material que no usaremos que está sobre el respaldo, comenzando a darle forma.

Para esto, en la sección de manufactura seleccionamos la opción Adaptive Clearing y lo configuramos. Seleccionamos la misma fresa con la que hemos estado trabajando (flat end mill de 6mm de diámetro). Esta vez el “machining boundary” será el sketch que creamos con el área del respaldo, delimitando que solo se mecaniza dentro de este límite. Y finalmente seleccionamos que el optimal load sea de 3mm. EL mecanizado debe quedar así.

Pero en este punto surge un problema, y es que la fresa es demasiado corta para lo que estamos pidiendo, por lo que se genera un impacto cuando se está mecanizando.

En las imágenes se puede observar partes del proceso donde pasa esto y la fresa no es lo suficientemente larga para cumplir bien su función.

Para evitar esto, debemos alargar la fresa con la que estamos trabajando, de esta forma se evitará este contacto que puede terminar dañando tanto a la máquina como la madera. Para esto, debemos hacer seleccionar “tool library”, copiar la herramienta que estamos usando (flat end mill 6mm) y luego editarla.

Podemos cambiarle el nombre para diferenciarla de la normal y en “cutter” debemos modificar el largo de la fresa. Donde dice “length below holder”, cambiamos 22.5 por 40 mm. Como vemos en la siguiente imágen, el largo se actualiza y ahora es suficiente para que no choque con el material durante el mecanizado.

Con esto dejamos configurada la nueva fresa que utilizaremos para el proceso. Para activarla, presionamos click derecho en el proceso que tiene que cambiar la fresa, editar herramienta, y seleccionamos la nueva fresa creada con el nombre que le pusimos. Ahora nuestro proceso no tendrá el problema mencionado anteriormente.

Paso 3

El siguiente paso es hacer terminaciones paralelas. Este proceso se hace posterior al Adaptative Clearing para darle mayor detalle. Este proceso es básicamente pasar una fresa a lo largo y a lo ancho del respaldo para que la curva quede mejor hecha.

Para hacer esto seleccionamos “Parallel” en el menú de mecanizados 3D. Seleccionamos el mismo machining boundary que antes, el área del respaldo proyectado en la superficie y en el modelo seleccionamos el asiento, apagando la opción “include setup model”, así trabajara solo sobre el asiento y no sobre todo el material.

Con este movimiento, la fresa recorre el respaldo continuamente, dándole una mejor terminación.

Pero volvemos a tener el mismo problema que en el mecanizado pasado, la fresa no es lo suficientemente larga para hacer este proceso sin chocar con el material. Esto se ve en la parte inferior de la pantalla, donde en la línea de tiempo se ven partes rojas. Una forma de solucionarlo es volviendo a seleccionar la fresa editada de 40 mm flat end mill, pero para este tipo de terminaciones paralelas hay una fresa que es mejor y permite dar una curvatura aún más precisa. La mejor fresa para este tipo de procesos es la de punta redonda, ya que al tener esa forma, puede hacer una curvatura más lisa y constante.

Es por esto que para este proceso seleccionamos una nueva fresa “Ball end mill” de diámetro 6 mm y la editamos igual que la anterior, cambiando “length below holder” de 22.5 por 40 mm.

Así, nuestra nueva biblioteca de fresas quedará de esta forma, con una fresa de punta plana (flat end mill) y una de punta redonda (ball end mill), ambas de 6mm de diámetro y con una distancia bajo el sostenedor de 40mm para evitar el contacto del sostenedor con el material.

Paso 4

Este paso es bastante sencillo ya que es básicamente igual al anterior pero de manera perpendicular. Para no tener que crear un mecanizado desde 0, podemos duplicar el usado en el paso anterior, creando así, una copia del mecanizado de terminaciones paralelas.

Una vez que tenemos esta copia hecha, se pone la opcion para editar y en la sección “passes”, escribo que la dirección del paso (pass direction) tiene un ángulo de 90 grados. Además, disminuyó la distancia entre cada pasada, así la definición será más alta. Esto se hace cambiando el Stepover de 3 mm a 2 mm.

Teniendo las dos terminaciones paralelas, generamos una mejor definición y con mayor detalle, dejándolo con una curva bastante precisa.

Desbaste

Mecanizado 2D y 2.5D

Para entender el mecanizado 2.5D es importante saber como es el mecanizado 2D antes.

El mecanizado 2D, si hablamos de una fresadora de 3 ejes, es un trabajo que se realiza solo con 2 de los 3 ejes a la vez. En otras palabras, es una superficie de proyección de un plano en tercera dimensión, por lo que no tiene elementos que sobresalen.

Por otro lado, el mecanizado 2.5D si trabaja con elementos que sobresalen y solo permite una única elevación en el eje Z.

Finalmente, el mecanizado 3D es como la 2.5D pero puede trabajar en distintas elevaciones en el eje Z.

Tiempos de Mecanizado

El mecanizado de la primera cara toma 1:33:56 en hacerse. Por otra parte, el mecanizado de la segunda cara tiene un tiempo estimado de 1:31:51.

Esto es un largo tiempo y hay que considerar que es solo para el mecanizado del respaldo. Pero hay una opción de reducir este tiempo.

Una de las formas de reducir el tiempo de mecanizado es la función de clasificación axial, que permite dividir el mecanizado en zonas. Básicamente lo que hace esta función es reducir los recorridos en vacío, generando un mecanizado continuo.

Otra forma de disminuir el tiempo es en modo flanco, donde se pueden mecanizar los costados con un fresado lateral. Al hacerlo, el usuario puede elegir si desea utilizar el modo de fresado lateral para todos los cortes o solo para el último. Esta nueva función permite obtener un mejor sobreespesor para el acabado y, al no ser necesario el acabado previo, el tiempo de fabricación se reduce.

Con estos procesos es posible disminuir hasta un 25% el tiempo de mecanizado dependiendo del trabajo, por lo que es un ahorro significativo en el tiempo y puede significar un aumento en la productividad.
