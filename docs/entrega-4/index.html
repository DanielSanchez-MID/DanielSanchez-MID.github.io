Bonus Molde

Trabajando un molde

En esta ocasión se muestra el mecanizado de un molde en Fusion 360. Para esto, se usaron 2 tipos de mecanizado. Uno de desbaste (Adaptive Clearing) y otro paralelo para darle un mayor detalle.

Este proceso es muy similar al que vimos en la página donde se explica el mecanizado del respaldo de la silla Fresia.

El resultado físico de este proceso debería verse así:

Para esto, se trabajó en fusion 360 para posteriormente mecanizar en una fresadora CNC.

Paso 1

El primer paso es hacer una plancha al molde que sobresalga un poco en sus extremos, de esta forma tenemos el material para darle forma la molde.

Esto se hace creando un sketch en la base del molde un poco más grande que este y extruyendo, de esta forma queda todo el molde dentro.

Paso 2

El segundo paso es hacer un sketch en la parte superior de la plancha, proyectando el área del molde sobre este. Esto se hace con un sketch de tipo “project” y seleccionando el molde.

Paso 3

En este paso comenzamos a mecanizar. Para partir debemos hacer el desbaste, eliminando el material que no necesitamos y está sobre el molde. Para esto, nos vamos a manufacture y seleccionamos la opción Adaptive Clearing. Dentro de esta opción, ponemos como límite el segundo sketch que hicimos (área del molde proyectada) y seleccionamos como objeto el molde. La fresa utilizada para este proceso es una flat end mill de 6 mm de diámetro. El desbaste debe quedar algo como esto:

Paso 4

El siguiente paso es hacer las paralelas. Para este proceso es mejor utilizar la fresa de punta redonda (ball end mill) de 6 mm de diámetro. En este caso nos vamos a la opción Parallel (en menú manufacture) y volvemos a poner como objeto el molde. Con este proceso se genera mayor detalle en las curvas ya que pasa de manera paralela a lo largo del molde y con una distancia determinada por nosotros. La punta redonda siempre es mejor para este tipo de mecanizados.

Así debe quedar el primer proceso paralelo hecho. Luego, el segundo se puede hacer duplicando el primero y editando sus configuraciones. Una vez en sus configuraciones, debemos cambiar el ángulo en el que pasa (pass direction) y poner 90 grados. Así, le estamos diciendo que rote su trayectoria en 90 grados y la mantenemos sobre el molde. Esto se hace en esta parte.

Es importante destacar en este punto, que la distancia entre cada recorrido paralelo está directamente relacionado con el detalle en el molde, es decir, mientras menor sea la distancia entre los recorridos, mejor será la calidad del mecanizado. Es por eso, que en esta parte he puesto 0.5 mm (stepover). Al ser un modelo pequeño, para tener buena calidad la fresa debe pasar muy cerca en cada recorrido.

Luego de esto, el segundo fresado paralelo debe quedar así.

A continuación algunas imágenes de los procesos explicados anteriormente.

El último paso para terminar el molde, es hacer un contour 2D de su perímetro inferior. De esta forma, se separa el molde del resto del material y podemos removerlo para utilizarlo. En este punto es importante recordar que se deben colocar tabs alrededor del contour para evitar que el molde caiga antes de que se termine el proceso, ya que esto puede dañar tanto la máquina como el material. Con estos tabs se mantiene unido al resto del material pero permite desprenderse una vez fuera de la máquina fácilmente.

Con esto se da por finalizado el proceso del molde. una vez que se separa el molde del resto de elementos sirve para crear figuras con su forma.
