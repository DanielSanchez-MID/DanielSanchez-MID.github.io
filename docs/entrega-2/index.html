Mecanizado CNC: Silla Fresia

Encargo 2

Pasos en Fusion 360:

Una vuelta por el proceso de fabricación de la Silla Fresia.

Esta es una silla que presenta un diseño moderno y, en este caso, simple, haciendola fácil de armar localmente.

En esta publicación veremos el proceso por el que pasa esta silla para ser cortada y fabricada con una máquina generic 3 axis.

En este caso, la silla cuenta con 2 piezas por cada pata y 4 piezas que unen las patas con el asiento. Además, cuenta con 2 partes, que son con las que interactuamos nosotros, que son curvas, por lo que el proceso para fabricarlas cambia un poco. Estas piezas son el respaldo y el asiento.

A continuación se mostrará documentación sobre el proceso en software que se debe hacer para poder hacerla con un mecanizado en CNC.

Partes planas

Lo primero que se hace es separar cada parte (sin curvatura) de la silla y colocarlas en una plancha hecha por nosotros de medidas 1220mm x 1220mm y 15mm de alto. Una vez que tenemos la plancha y las piezas dentro de esta, tenemos que seleccionar la máquina que es una generic 3 axis (esto se hace en setup). Debe quedar algo así:

A continuación debemos determinar los tipos de corte que queremos hacer. Para esto vamos al menú de MANUFACTURE.

El primero es el de cada espacio que tiene las piezas para el encaje. Este se hace con un contour 2D, seleccionando el borde inferior de estos espacios pequeños.Para esto se usa una fresa de 6mm de diámetro. También es necesario poner que el maximun roughing stepdown sea de 3mm, de esta forma se cuida la máquina y el CNC, evitando forzar demasiado la máquina. Esta opción se debe seleccionar en todas las operaciones a continuación. Lo que genera este corte es lo siguiente.

Una vez que tenemos ese primer corte, debemos hacer las rebajas a ciertas piezas como las patas donde tienen un grosor menor a la plancha de CNC. Para esto, se utiliza 2D pocket y se selecciona la cara de esta parte más delgada, así se rebajarán estas y quedarán con los cortes correctos para unir las piezas. El resultado de esta operación es el siguiente:

Finalmente, se debe hacer con un contour 2D, seleccionando todo el contorno inferior de las piezas. Así, la máquina cortará la forma de cada pieza. Para que no se caiga la pieza y evitar problemas hay que poner tabs de 6mm. No olvidar seleccionar un maximun roughing stepdown de 3mm.

Partes curvas (asiento)

En el caso del asiento, que es curvo, se trabaja con una madera plana también, y para poder generar esa curvatura una vez sacada de la máquina se deben crear vectores para que la máquina pase por ahí y permita darle forma posteriormente.

Primero se crea una plancha (de CNC) de 9mm de alto y área que cubra la pieza y se repite el mismo proceso de las piezas anteriores con el contour 2D (no olvidar tabs de 6mm). Luego para estos vectores usamos una nueva función en el menú de 2D. Seleccionamos TRACE y aplicamos a los vectores en el asiento. Para que estos vectores no atraviesen el material completo seleccionamos en “stock to leave: 4.5mm”. De esta forma se corta hasta la mitad y permite darle una curvatura luego. El asiento se debería quedar de esta manera.

En el video a continuación se ve cómo se cortan estos vectores para darle flexibilidad al asiento y como la máquina no atraviesa todo el CNC, sino solo 4.5mm.

Luego de este proceso para las distintas piezas, las simulaciones de la máquina se verán así, siguiendo el orden mencionado y dejando las piezas listas para ser sacadas.

Máquina CNC

Un detalle que no ha sido explicado es el por qué de estas “orejas” que se generan cuando se corta un área en el interior del material. Es un espacio que queda en los encajes de la silla. Se puede ver a continuación.

Estas “orejas”, se forman ya que, al ser la fresa una punta en rotación, no es capaz de hacer cortes en 90 grados cuando trabaja áreas dentro del material. Al rotar la fresa, siempre generará una figura curva al final, por lo que no es capaz de hacer esquinas en 90 grados. Pero como se necesita la totalidad del cuadrado (incluyendo las esquinas) para encajar las piezas, la fresa puede pasarse un poco del área determinada, exactamente la mitad de su diámetro, para así dejar la totalidad del área que se quería, cortada.

Como se ve en la imágen anterior, si no se hicieran estas “orejas” en el mecanizado, no se cubrirán las esquinas de la figura deseada (cuadrado) y por lo tanto no se podrían encajar las piezas.

En la parte de abajo de la imágen, se ve hasta donde llega la fresa en cada “oreja”, cubriendo así, toda el área deseada.

Esto no pasa cuando se mecaniza una caja exterior, ya que la fresa no puede hacer dos recorridos que generen un angulo externo de 90 grados.
